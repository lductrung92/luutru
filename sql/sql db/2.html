Câu lệnh ALTER TABLE được sử dụng để thêm, xóa, hoặc sửa đổi các cột trong một bảng hiện có.

Câu lệnh ALTER TABLE cũng được sử dụng để thêm và bỏ các ràng buộc khác nhau trên một bảng hiện có.

ALTER TABLE - Thêm cột

Để thêm một cột trong một bảng, sử dụng cú pháp sau:

    ALTER TABLE table_name
    ADD column_name datatype;

ALTER TABLE - Xóa cột

Để xóa một cột trong một bảng, sử dụng cú pháp sau:

    ALTER TABLE table_name
    DROP COLUMN column_name;

ALTER TABLE - ALTER/MODIFY COLUMN

Để thay đổi kiểu dữ liệu của một cột trong một bảng, sử dụng cú pháp sau:

SQL Server/MS Access:

    ALTER TABLE table_name
    ALTER COLUMN column_name datatype;

SQL/Oracle:

    ALTER TABLE table_name
    MODIFY COLUMN column_name datatype;

Ví dụ: Có bảng "Persons":

ID LastName FirstName Address City
1 Hansen Ola Timoteivn 10 Sandnes
2 Svendson Tove Borgvn 23 Sandnes
3 Pettersen Kari Storgt 20 Stavanger

Bây giờ chúng ta muốn thêm một cột có tên "DateOfBirth" trong bảng "Persons".

Ta sử dụng câu lệnh SQL sau:

    ALTER TABLE Persons
    ADD DateOfBirth date;

Kết quả bảng "Persons" bây giờ:

ID LastName FirstName Address City DateOfBirth
1 Hansen Ola Timoteivn 10 Sandnes  
2 Svendson Tove Borgvn 23 Sandnes  
3 Pettersen Kari Storgt 20 Stavanger  

Tiếp theo, chúng ta muốn xóa cột có tên "DateOfBirth" trong bảng "Persons".

Ta sử dụng câu lệnh SQL sau:

    ALTER TABLE Persons
    DROP COLUMN DateOfBirth;

Kết quả bảng "Persons" sẽ có dữ liệu như ban đầu
