Mục Lục:

  • Create DB
  • Drop DB
  • Create Table
  • Drop Table

1. Create DB: được sử dụng để tạo ra một cơ sở dữ liệu SQL mới.

Cú pháp: 

        CREATE DATABASE databasename;

Ví dụ: Câu lệnh SQL sau tạo một cơ sở dữ liệu gọi là "testDB":

        CREATE DATABASE testDB;

2. Drop DB: được sử dụng để xóa một cơ sở dữ liệu SQL hiện có.

Cú pháp: 

        DROP DATABASE databasename;

Ví dụ: Câu lệnh SQL sẽ xóa cơ sở dữ liệu hiện có "testDB":

        DROP DATABASE testDB;

3. Create Table: được sử dụng để tạo một bảng mới trong cơ sở dữ liệu.

Cú pháp: 

        CREATE TABLE table_name (
            column1 datatype,
            column2 datatype,
            column3 datatype,
           ....
        );

Ví dụ: tạo một bảng "Persons" có chứa năm cột: PersonID, LastName, FirstName, Address và City:

        CREATE TABLE Persons (
            PersonID int,
            LastName varchar(255),
            FirstName varchar(255),
            Address varchar(255),
            City varchar(255)
        );

Tạo bảng sử dụng bảng khác

Một bản sao của một bảng hiện có có thể được tạo ra bằng cách kết hợp câu lệnh CREATE TABLE và câu lệnh SELECT.

Cú pháp: 

        CREATE TABLE new_table_name AS
        SELECT column1, column2,...
        FROM existing_table_name
        WHERE ....;

4. Drop Table: được sử dụng xóa một bảng hiện có trong cơ sở dữ liệu.

Cú pháp: 

        DROP TABLE table_name;

Ví dụ: Câu lệnh SQL sẽ xóa bảng hiện có "Shippers":

        DROP TABLE Shippers;

TRUNCATE TABLE: Câu lệnh TRUNCATE TABLE được sử dụng để xóa dữ liệu bên trong một bảng.

Cú pháp: 

        TRUNCATE TABLE table_name;
