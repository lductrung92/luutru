Bài hôm nay mình sẽ đi giới thiệu các lệnh: INSERT INTO, UPDATE, DELETE trong SQL

Mục Lục:

  • INSERT INTO
  • UPDATE
  • DELETE

1. INSERT INTO

Câu lệnh INSERT INTO được sử dụng để chèn các bản ghi mới vào một bảng.

Cú pháp:

Có thể viết câu lệnh INSERT INTO theo hai cách.

        INSERT INTO table_name (column1, column2, column3, ...)
        VALUES (value1, value2, value3, ...);

Nếu bạn đang thêm các giá trị cho tất cả các cột của bảng, bạn không cần chỉ định tên cột trong truy vấn SQL. Tuy nhiên, đảm bảo thứ tự của các giá trị đúng theo thứ tự như các cột trong bảng. Cú pháp INSERT INTO sẽ như sau:

        INSERT INTO table_name
        VALUES (value1, value2, value3, ...);

Ví dụ: Câu lệnh SQL sau chèn một bản ghi mới trong bảng "Customers":

        INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country)
        VALUES ('Cardinal', 'Tom B. Erichsen', 'Skagen 21', 'Stavanger', '4006', 'Norway');

2. UPDATE

Câu lệnh UPDATE được sử dụng để sửa đổi các bản ghi hiện có trong một bảng.

Cú pháp: 

        UPDATE table_name
        SET column1 = value1, column2 = value2, ...
        WHERE condition;

Ví dụ: Câu lệnh SQL sau đây cập nhật cho khách hàng đầu tiên (CustomerID = 1) với một ContactName và một City mới.

        UPDATE Customers
        SET ContactName = 'Alfred Schmidt', City= 'Frankfurt'
        WHERE CustomerID = 1;

Ví dụ: Câu lệnh SQL sau đây sẽ cập nhật ContactName "Juan" cho tất cả các bản ghi ở đó quốc gia là "Mexico":

       UPDATE Customers
       SET ContactName='Juan'
