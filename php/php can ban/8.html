Mục Lục:

  • Lệnh if
  • Lệnh if ... else
  • Lệnh if ... elseif .... else

1. Lệnh if

Cú pháp

        if (condition) {
            // to do something
        }
    
        $t = date("H");

        if ($t < "20") {
            echo "Have a good day!";
        }

2. Lệnh if ... else

Cú pháp

        if (condition) {
           // code thực hiện khi điện kiện là đúng
        } else {
            // code thực hiện khi điện kiện là sai
        }
    
        $t = date("H");

        if ($t < "20") {
            echo "Have a good day!";
        } else {
            echo "Have a good night!";
        }

3. Lệnh if ... elseif .... else

Cú pháp

        if (condition) {
            // code thực hiện khi điện kiện là đúng
        } elseif (condition) {
            // code thực hiện khi điện kiện là đúng
        } else {
            // code thực hiện khi điện kiện là sai
        }
    
        $t = date("H");

        if ($t < "10") {
            echo "Have a good morning!";
        } elseif ($t < "20") {
            echo "Have a good day!";
        } else {
            echo "Have a good night!";
        }
